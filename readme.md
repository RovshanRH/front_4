# Магазин Электроники

## Введение

Это учебный мини-магазин электроники с каталогом товаров.

Приложение позволяет:
- просматривать карточки товаров;
- открывать подробную информацию в модальном окне;
- добавлять новый товар;
- редактировать существующий товар;
- удалять товар.

Фронтенд построен на React (Vite), бэкенд - на Node.js + Express.
Данные хранятся в памяти сервера (массив в `server/products.js`), поэтому после перезапуска сервера изменения сбрасываются.

## Структура проекта

- `client/` - клиентская часть (React + Vite + SCSS).
- `server/` - API-сервер (Express, CORS, Swagger).
- корневой `package.json` - общий запуск клиента и сервера одной командой.

## Инструкция по запуску

### 1) Установка зависимостей

В корне проекта:

```bash
npm install
```

В папке клиента:

```bash
npm --prefix client install
```

В папке сервера:

```bash
npm --prefix server install
```

### 2) Запуск в режиме разработки

Из корня проекта:

```bash
npm run dev
```

Эта команда одновременно запускает:
- сервер на `http://localhost:4000`;
- клиент (Vite) на `http://localhost:5173`.

### 3) Полезные адреса

- Клиент: `http://localhost:5173`
- API товаров: `http://localhost:4000/api/products`
- Swagger UI: `http://localhost:4000/api-docs`

## Доступные команды

Корень проекта:

```bash
npm run dev      # запустить клиент и сервер вместе
npm run client   # запустить только клиент
npm run server   # запустить только сервер
```

Клиент (`client`):

```bash
npm run dev
npm run build
npm run preview
```

Сервер (`server`):

```bash
npm run dev
npm run start
```
